##CSS의 기초 - 단어 간격(word-spacing)

단어와 단어 사이의 간격을 지정합니다. 

### word-spacing
- 기본값 : 부모상속됨
- 자식상속 : 상속됨
- 작성방법 : `word-spacing: normal|length;`

#### 주로 쓰이는 값
값 | 설명
---| ----
length | [단위](../step1/11_Unit.md)로 단어 간격에 추가하여 반영합니다.

#### 거의 안쓰이는 값
값 | 설명
---| ----
normal | 폰트가 가진 간격으로 표현됩니다. 0 와 같습니다.

#### 실무 포인트
- 자간(letter-spacing)과 중첩적용됩니다.
