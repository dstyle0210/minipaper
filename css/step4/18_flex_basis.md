##CSS의 기초 - 플랙스아이템 기본값 설정(flex-basis)

플랙스 아이템 의 디렉션에 따른 사이즈를 기본설정합니다.

### flex-basis
- 기본값 : auto
- 상속성 : 없음
- 작성방법 : `flex-basis: number|auto|initial|inherit;`

#### 주로쓰이는 값
값 | 설명
---| ----
number | 부모요소의 디렉션에 따른 아이템의 기본사이즈를 [단위]()로 지정합니다.


#### 꼭 기억하세요.
- 본 속성은 해당 디렉션에 따른 사이즈값(width|height)을 무시하고 적용됩니다.
