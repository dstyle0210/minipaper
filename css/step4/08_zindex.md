##CSS의 기초 - 제트인덱스(z-index)

포지션된 요소중 "stactic" 이 아닌 요소의 공중에 뜬 높이를 지정합니다.

### z-index
- 기본값 : auto
- 상속성 : 없음
- 작성방법 : `z-index: auto|number|initial|inherit;`

#### 주로쓰이는 값 
값 | 설명
---| ----
number | 띄울 높이를 숫자형으로 지정합니다.(음수도 가능)
 
 
#### 꼭 기억하세요.
- "z-index"을 음수로 적용하면, 아래로 가라앉아서 사라지게 되면서, 포커스도 안잡히게 됩니다.
