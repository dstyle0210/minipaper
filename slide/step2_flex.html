플랙스 레이아웃이란?

요소의 방향축(axis)을 조정하여 레이아웃하는 방법입니다.

  • 현재 플랙스 레이아웃은 모바일 브라우저 및 최신 브라우저에서만 정상적으로 동작합니다.
  • 컨테이너(container)는 플랙스레이아웃의 부모요소 이면서, 전체영역 입니다.
  • 아이템(item)은 플랙스레이아웃의 자식요소 입니다.
  • 흐름방향(main axis)는 플랙스레이아웃의 흐름(Flow)의 기준입니다.
  • 흐름방향(main axis)는 main-start(흐름방향 시작) 와 main-end(흐름방향 끝)으로 나뉘어집니다.
  • 정렬방향(cross axis)은 플랙스레이아웃의 정렬(Align) 방향입니다.
  • 정렬방향(cross axis)은 cross-start(정렬방향 시작) 와 cross-end(정렬방향 끝)으로 나뉘어집니다.

컨테이너 : display:flex;

플랙스 레이아웃을 사용하겠다 라는 선언에 대해 설명합니다.

  • 과거에는 "display:box" , "display:flexbox" , "display:flex" 이 3가지는 시기가 틀릴뿐 동일 레이아웃 선언이며, 현재는 "display:flex"로 통합되었습니다.
예제보기

컨테이너 : flex-direction

컨테이너의 흐름방향(main axis)을 설정합니다.

  • 기본값 : row
  • 상속성 : 없음
  • 작성방법 : flex-direction: row|row-reverse|column|column-reverse|initial|inherit;
row "좌->우" 로 흐름방향(main axis)을 설정합니다.
row-reverse "우->좌" 로 흐름방향(main axis)을 설정합니다.
column "상->하" 로 흐름방향(main axis)을 설정합니다.
column-reverse "하->상" 으로 흐름방향(main axis)을 설정합니다.
예제보기

컨테이너 : flex-wrap

컨테이너의 흐름방향(main axis)에 따른 줄바꿈 여부를 설정합니다.

  • 기본값 : nowrap
  • 상속성 : 없음
  • 작성방법 : flex-wrap: nowrap|wrap|wrap-reverse|initial|inherit;
nowrap 아이템에 대해 줄바꿈 처리하지 않습니다. 아이템의 사이즈가 강제 조정됩니다.
wrap 아이템이 컨테이너의 정렬기준(cross start -> cross end)로 줄바꿈 배치 됩니다.
wrap-reverse 아이템이 컨테이너의 역 정렬기준(cross end -> cross start)로 줄바꿈 배치 됩니다.
예제보기

컨테이너 : flex-flow

flex_direction와 flex_wrap을 단축 선언합니다.

  • 기본값 : row nowrap
  • 상속성 : 없음
  • 작성방법 : flex-flow: flex-direction flex-wrap|initial|inherit;

컨테이너 : justify-content

컨테이너의 흐름방향(main axis)에 따른 아이템 배치방법을 설정합니다.

  • 기본값 : flex-start
  • 상속성 : 없음
  • 작성방법 : justify-content: flex-start|flex-end|center|space-between|space-around|initial|inherit;
flex-start 아이템을 흐름방향 시작점(main start)을 기준으로 정렬합니다.
flex-end 아이템을 흐름방향 종료점(main end)을 기준으로 정렬합니다.
center 아이템을 흐름방향(main axis) 중간점을 기준으로 정렬합니다.
space-between 아이템을 흐름방향 시작점(main-start) 와 종료점(main end)을 기준하여 아이템을배치합니다.
space-around 아이템을 흐름방향 시작점(main-start) 와 종료점(main end)을 기준하여 공간을배치합니다.

컨테이너 : align_items

컨테이너의 정렬방향(cross axis)에 따른 정렬(align)를 설정합니다.

  • 기본값 : stretch
  • 상속성 : 없음
  • 작성방법 : align-items: stretch|center|flex-start|flex-end|baseline|initial|inherit;
stretch 컨테이너의 정렬기준(cross axis) 사이즈에 아이템들의 사이즈를 맞춥니다.
center 아이템 사이즈 번화없이 정렬기준(cross axis)의 중간점을 기준으로 배치합니다.
flex-start 아이템 사이즈 번화없이 정렬기준 시작점(cross start)을 기준으로 배치합니다.
flex-end 아이템 사이즈 번화없이 정렬기준 종료점(cross end)을 기준으로 배치합니다.
baseline 아이템 사이즈 번화없이 컨테이너의 baseline을 기준으로 배치합니다.
연습하기

컨테이너 : align_content

컨테이너의 정렬방향(cross axis)에 따른 줄바꿈시 정렬방식을 설정합니다.

  • 기본값 : stretch
  • 상속성 : 없음
  • 작성방법 : align-content: stretch|center|flex-start|flex-end|space-between|space-around|initial|inherit;
stretch 줄바꿈이 일어날경우, 컨테이너의 정렬기준(cross axis) 사이즈에 아이템들의 사이즈를 맞춥니다.
center 줄바꿈이 일어날경우, 아이템 사이즈 번화없이 컨테이너의 정렬기준(cross axis) 중간점에 아이템을 모아서 배치합니다.
flex-start 줄바꿈이 일어날경우, 아이템 사이즈 번화없이 컨테이너의 정렬기준 시작점(cross start)에 아이템을 모아서 배치합니다.
flex-end 줄바꿈이 일어날경우, 아이템 사이즈 번화없이 컨테이너의 정렬기준 종료점(cross end)에 아이템을 모아서 배치합니다.
space-between 줄바꿈이 일어날경우, 아이템 사이즈 번화없이 컨테이너의 정렬기준 시작점(cross-start) 와 종료점(cross end)을 기준하여 아이템을배치합니다.
space-around 줄바꿈이 일어날경우, 아이템 사이즈 번화없이 컨테이너의 정렬기준 시작점(cross-start) 와 종료점(cross end)을 기준하여 공간을배치합니다.
  • 줄바꿈이 일어나야지만 하므로, flex-wrap:nowrap 일경우, 본 속성은 무시됩니다.
  • 본 속성이 선언 될 경우, align-items 속성은 무시됩니다.

아이템 : flex_basis

흐름방향(main axis)에 대한 아이템의 기본값(basis)을 적용합니다.

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : flex-basis: length|auto|initial|inherit;
auto 아이템의 기본값을 자동으로 가져옵니다.
length 아이템의 기본값을 단위로 지정합니다.
  • "auto"의 경우에는 아이템의 사이즈의 기준이 베이스가 됩니다.
연습하기

아이템 : flex-grow

컨테이너에 아이템이 배치되고 남은 공간을 아이템에 분배하는 비(ratio)을 설정합니다.

  • 기본값 : 0
  • 상속성 : 없음
  • 작성방법 : flex-grow: number|initial|inherit;
length 남는 공간에 대해 아이템이 가져갈 비(ratio)을 양수로 표현합니다.
  • 기본값인 "0" 은 분배 받지 않겠다. 라는 의미입니다.
  • 남는 공간이 120px이고, 아이템들의 flex-grow값의 총합이 3 이라면, flex-grow:1은 40px 씩 늘어납니다.
연습하기

아이템 : flex-shrink

컨테이너에 아이템이 배치되고 넘쳐진 공간을 아이템에 분배하는 비(ratio)을 설정합니다.

  • 기본값 : 1
  • 상속성 : 없음
  • 작성방법 : flex-shrink: number|initial|inherit;
length 넘쳐진 공간에 대해 아이템이 가져갈 비(ratio)을 양수로 표현합니다.
  • "0" 으로 선언할경우, 분배받지 않겠다(줄어들지 않겠다) 라는 의미 입니다.
  • 넘쳐진 공간이 90px이고, 아이템들의 flex-shrink 값의 총합이 3 이라면, flex-shrink:1은 30px 씩 줄어듭니다.
연습하기

아이템 : flex

flex-grow , flex_shrink , flex_basis을 단축선언합니다.

  • 기본값 : 0 1 auto
  • 상속성 : 없음
  • 작성방법 : flex: flex-grow flex-shrink flex-basis|auto|initial|inherit;

아이템 : align-self

아이템 개별의 정렬방향(cross axis)을 설정합니다.

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : align-self: auto|stretch|center|flex-start|flex-end|baseline|initial|inherit;
auto 컨테이너의 "align-items" 속성을 상속받습니다.
stretch 컨테이너의 정렬기준(cross axis) 사이즈에 아이템의 사이즈를 맞춥니다.
center 아이템 사이즈 번화없이 정렬기준(cross axis)의 중간점을 기준으로 배치합니다.
flex-start 아이템 사이즈 번화없이 정렬기준 시작점(cross start)을 기준으로 배치합니다.
flex-end 아이템 사이즈 번화없이 정렬기준 종료점(cross end)을 기준으로 배치합니다.
baseline 아이템 사이즈 번화없이 컨테이너의 baseline을 기준으로 배치합니다.
  • 컨테이너의 "align-items" 속성은 기본값이 "stretch" 입니다.
연습하기

아이템 : order

흐름방향(main axis)상 아이템의 우선순위를 지정합니다.

  • 기본값 : 0
  • 상속성 : 없음
  • 작성방법 : order: number|initial|inherit;
number 흐름방향(main axis)상 아이템의 우선순위를 지정합니다.
  • 음수도 적용 가능합니다.
