박스모델(Box Model)

CSS의 기본구조인 박스모델에 대해서 알아봅니다.

Content 내부의 내용을 말합니다. 물론 다른 엘리먼트가 들어올수도 있습니다.
Padding 컨덴츠와 보더사이의 간격입니다. 배경의 영향을 받습니다.
border 컨덴츠와 패딩을 포함한 영역의 내곽선입니다. 가로값의 영향을 받습니다.
Margin 보더와 외부 엘리먼트의 간격입니다. 배경의 영향을 받지않습니다.
outline 컨덴츠와 패딩 및 보더를 포함한 포함한 외곽선입니다. 가로값의 영향을 받지 않습니다.

가로값(width)

요소의 가로값을 지정합니다.

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : width: length | auto;

  • min-width

    min-width: length | auto;

  • max-width

    max-width: length | auto;

  • 작성방법 : height: length | auto;

    max-height: length | auto;

보더(border)

컨덴츠와 패딩을 포함한 영역의 내곽선입니다.

  • 기본값 : none
  • 자식상속 : 안함
  • 작성방법 : border: border-width border-style border-color;

  • border-top

    border-top: border-width border-style border-color;

  • border-left

    border-left: border-width border-style border-color;

  • border-bottom

    border-bottom: border-width border-style border-color;

  • border-right

    border-right: border-width border-style border-color;

  • border-width

    border-width: medium | thin | thick | length | initial | inherit;

  • border-style

    border-style:none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset | initial | inherit;

  • border-color

    border-color: color|transparent|initial|inherit;

  • border-radius

    border-radius: 1/4 length | % / 1/4 length | % | initial | inherit;

박스사이징(Box Sizing)

사이즈 부분에 대한 포함범위를 설정합니다.

  • 기본값 : content-box
  • 상속성 : 없음
  • 작성방법 : box-sizing: content-box|border-box|initial|inherit;
content-box 컨덴츠 영역만 박스사이즈(가로값,세로값)로 인정합니다.
border-box 컨덴츠 및 패딩, 보더 영역까지 박스사이즈(가로값,세로값)로 인정합니다.

    background-image: url|none|initial|inherit;

    background-position: xpos ypos;

  • background-size

    background-size: auto|length|cover| contain|initial|inherit;

  • background-repeat

    background-repeat: repeat|repeat-x|repeat-y|no-repeat|initial|inherit;

  • background-origin

    background-origin: padding-box|border-box|content-box|initial|inherit;

  • background-clip

    background-clip: border-box|padding-box|content-box|initial|inherit;

  • background-attachment

    background-attachment: scroll|fixed|local|initial|inherit;

디스플레이(display)

요소의 박스모델 속성을 변경합니다.

  • 기본값 : 엘리먼트형태 상속
  • 상속성 : 없음
  • 작성방법 : display: inline | block | inline-block | none | flex | table | table-cell;
값 설명
inline 인라인요소로 적용합니다.
block 블럭요소로 적용합니다.
inline-block 인라인-블럭 요소로 적용합니다.
none 화면 상으로 완전히 사라지도록 처리합니다.
flex CSS3의 flex 레이아웃 구조의 컨테이너로 적용합니다. (기본은 block속성을 따라갑니다.)
table 테이블 요소로 적용합니다. (기본은 inline-block을 따라갑니다.)
table-cell 테이블 요소의 "td" 또는 "th" 요소로 적용합니다.

비지빌리티(visibility)

요소의 보여짐 여부를 결정합니다.

  • 기본값 : visible
  • 상속성 : 없음
  • 작성방법 : visibility: visible|hidden|collapse|initial|inherit;
값 설명
visible 화면상 보여지도록 합니다.
hidden 화면상 보여지지 않도록 합니다.
collapse 테이블요소 한정으로 보이지 않도록 처리합니다. 다른요소에 사용하면 "hidden"과 같습니다.
  • "display"와 의 차이점을 꼭 기억하세요.
  • display: 요소의 속성을 바꾼다. / visibility: 보이고, 안보이고만 처리한다.
