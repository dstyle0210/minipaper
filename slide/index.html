상속성(Cascading)

물처럼 흘러내려서 CSS?

상속성 제외

텍스트 관련 속성(text-align, font-size 등)은 상속

박스모델 관련 속성(width,height)는 영향은 줄수 있지만 상속안됨

우선순위(Priority Rules)

  1. "!important" 구문
  2. inline style 선언 (<E style=""></E>)
  3. id (#) 로 선언
  4. class (.) 로 선언
  5. 엘리먼트로 선언
  6. 유니버셜 (*) 선언
  7. 작성순서에 따른 오버라이딩
  8. native style sheet (네이티브 스타일 시트) | 상속성에 따라 상속된 선언
