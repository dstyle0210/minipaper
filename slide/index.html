CSS("Cascading Style Sheets")란?

상속성(Cascading)

물처럼 흘러내려서 CSS?

상속성 제외

텍스트 관련 속성(text-align, font-size 등)은 상속

박스모델 관련 속성(width,height)는 영향은 줄수 있지만 상속안됨

우선순위(Priority Rules)

  • 1. "!important" 구문
  • 2. inline 직접선언 ( <E style=""></E> )
  • 3. id (#) 로 선언
  • 4. class (.) 로 선언
  • 5. 엘리먼트로 선언
  • 6. 유니버셜 (*) 선언
  • 7. 작성순서에 따른 오버라이딩
  • 8. native style sheet (네이티브 스타일 시트) | 상속성에 따라 상속된 선언

오버라이딩

우선순위가 같다면, 나중에 선언된 속성이 적용됩니다.

작성법

셀렉터(선택자) 란?

DOM 구조에서 스타일을 적용할 엘리먼트를 찾아가는 지시자.

퀴즈퀴즈

아래 로고는 자바스크립트 라이브러리 인데요. 어떤 라이브러리인지 아시는분?

퀴즈퀴즈 - 정답

jQuery의 선택자(셀렉터) 프로젝트명입니다.

sizzlejs

셀렉터(Selector) - Basic

  • Universal Selector (*)

    *{color:red;}

  • Element Selector (E)

    span{color:red;}

  • id Selector(#)

    #id{color:red;}

  • Class Selector(.)

    .class{background-color:gray;}

  • Group Selector(,)

    h1 , #id , .class{color:red; }

셀렉터(Selector) - Attribute

  • [attribute]

    [class]{color:red;}

  • [attribute=value]

    [class="test"]{color:red;}

  • [attribute|=value]

    [class|="test"]{color:red;}

  • [attribute^=value]

    [class^="test"]{color:red;}

  • [attribute~=value]

    [class~="test"]{color:red;}

  • [attribute*=value]

    [class*="test"]{color:red;}

  • [attribute$=value]

    [class$="test"]{color:red;}

셀렉터(Selector) - Search

  • inside Selector ( ) // 빈칸

    h1 span{border:1px solid #000;}

  • Child Selector(>)

    h1>span{background-color:gray}

  • After Selector(+)

    h1+span{width:500px;}

  • Preceded Selector(~)

    h1~span{width:300px}

가상(Pseudo) 셀렉터 란?

Pseudo Class(피도 클래스)라는것은 엘리먼트가 특정한 상태일때 을 지칭합니다.

Pseudo Element(피도 엘리먼트)는 가상의 엘리먼트를 생성해서 속성을 지정합니다.

단독으로 사용할수는 있으나, 추천하지는 않으므로, 언제나 셀렉터에 연결하여 선언됩니다.

가상(Pseudo) 진짜많이쓰는 클래스

  • Selector:checked

    input[class=checkbox]:checked{}

  • Selector:disabled

    input[class=checkbox]:disabled{}

  • Selector:focus

    input:focus{color:red;}

  • Selector:hover

    a:hover{}

  • Selector:first-child

    div p:first-child{}

  • Selector:nth-child(number | odd | even | n)

    div p:nth-child(2){}

  • Selector:read-only

    input:read-only{}

가상(Pseudo) 그냥 쫌 쓰는 클래스

  • Selector:active

    a:active{color:red;}

  • Selector:link

    a:active{color:red;}

  • Selector:visited

    a:active{color:red;}

  • Selector:last-child

    div p:last-child{}

  • Selector:not(Selector)

    input:not(:checked){}

  • Selector:nth-last-child(number | odd | even | n)

    div p:nth-last-child(2){}

가상(Pseudo) 거의 몰라도 되는 클래스

  • Selector:empty
  • Selector:first-of-type(number)
  • Selector:last-of-type(number)
  • Selector:nth-of-type(number)
  • Selector:in-range
  • Selector:out-of-range
  • Selector:valid
  • Selector:invalid
  • Selector:lang(languagecode)
  • Selector:only-of-type
  • Selector:only-child
  • Selector:required
  • Selector:optional
  • Selector:read-write
  • Selector:root
  • Selector:target

가상(Pseudo) 엘리먼트

  • Selector::before or Selector:before

    .clearfix::before{}

  • Selector::after or Selector:after

    .clearfix::after{}

가상(Pseudo) 몰라도 되는 엘리먼트

  • Selector::first-line
  • Selector::first-letter
  • Selector::selection
