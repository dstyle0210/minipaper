CSS란?

CSS(Cascading Style Sheet)는 HTML문서를 꾸미는 언어입니다.

작성법(Syntax)

셀렉터 부분이 있고, 중괄호({}) 안에 속성(Property)와 값(Value)으로 이루어져 있습니다.

적용방법

HTML에 CSS을 입히는 방법은 총 4가지가 있습니다.

<link> <link rel="stylesheets" href="CSS경로" />
<style> <style> CSS 선언 </style>
style="CSS선언" <E style="CSS선언">
@import "CSS경로"; @import "CSS경로"
예제 : sample_css_beginner.html

셀렉터(Selector) : 선택

CSS의 셀렉터를 이용해서, HTML태그(엘리먼트)를 선택합니다.

  • Universal Selector (*)

    *{property:value;}

  • Element Selector (E)

    E{property:value;}

  • id Selector(#)

    #id{property:value;}

  • Class Selector(.)

    .className{property:value;}

  • Group Selector(,)

    E , #id , .className{property:value;}
  • HTML태그의 "id" 와 대입 되는것이 "#" 이고, "class"와 대입되는것이 "." 입니다.
  • class로 선택할때는 class 안에 여러개의 class 값이 들어갈 수 있습니다.

셀렉터(Selector) : 검색

CSS의 셀렉터로 HTML태그를 검색해서 선택합니다.

  • inside Selector ( )

    E E{property:value;}

  • Child Selector(>)

    E>E{property:value;}

  • Next Selector(+)

    E+E{property:value;}

  • After Selector(~)

    E~E{property:value;}

컬러(Color)

CSS에서 적용할 색상 작성법 입니다.

  • HEX(Hexadecimal)

    E{property:#RRGGBB}

  • RGB , RGBA

    E{property:rgb(0,0,0) | rgba(0,0,0,0);}

  • 컬러이름.(red , blue 등)

    E{property:ColorName}
  • 대부분은 색상을 적용할때는 "HEX"로 #영문소문자6자리 로 표기합니다.

단위(Unit)

CSS에서 사용되는 단위의 종류입니다.

  • Pixel(px)

    E{property:numberpx;}

  • Percentage(%)

    E{property:number%;}

  • EM(em)

    E{property:numberem;}
  • 단위는 앞으로 나올 설명에서 "length"와 같아요.
  • "%"는 부모 HTML태그의 사이즈(가로,세로)를 기준으로 적용됩니다.(100% == 부모의 사이즈)
  • "EM"은 부모 HTML태그의 글자크기("font-size")을 기준으로 적용됩니다.(1em == 부모의 font-size)

우선순위(Priority Rules)

셀렉터에는 우선순위가 있어요.

  • 1. inline 직접선언 ( <E style=""></E> )
  • 2. id (#) 로 선언
  • 3. class (.) 로 선언
  • 4. 엘리먼트로 선언
  • 5. 유니버셜 (*) 선언
  • 총합이 아니라 검색단계의 우선순위에 따라 적용됩니다.
  • 우선순위가 같으면, 나중에 선언된 속성이 적용됩니다.

강제적용(!important)

우선순위가 제일 높은 "!important"입니다.

E{property:value !important;} , <E style="property:value !important;">
  • 값 뒤에 세미콜론(;)없이 "!important"을 입력해야 합니다.

가상 셀렉터(Pseudo Classes)

특정조건일때 선택하는 방법을 알아봅니다.

  • :checked

    E:checked{property:value;}

  • :disabled

    E:disabled{property:value;}

  • :focus

    E:focus{property:value;}

  • :hover

    E:hover{property:value;}

  • :first-child

    E:first-child{property:value;}

  • :nth-child(number|odd|even|n)

    E:nth-child(n){property:value;}
  • 실무에서 많이 사용하는 가상 셀렉터만 정리 했고, 실제로는 엄청 많이 있어요.[참고]

가상 셀렉터(Pseudo Classes) : nth-child(n)

가상셀렉터 중 "nth-child"에 대해서 연구.

:nth-child(n)
  • "an+b"를 이해하세요.
  • "2n"은 "even"과 같고, "2n+1" 은 "odd"와 같아요.
  • "번째"를 선택하겠다 라는 개념이 중요합니다. ( 1st, 2nd, 3rd, 4th, ..., nth )
  • IE8에서 호환성이 안되기 때문에 주의해야 합니다.
