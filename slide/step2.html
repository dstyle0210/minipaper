1주차 수업정리

셀렉터(Selector)

  • Universal Selector (*) : 전체 선택

    *{속성:값}

  • Element Selector (E) : HTML 엘리먼트 이름으로 선택

    E{속성:값;}

  • id Selector(#) : id 선택

    #id{속성:값;}

  • Class Selector(.) : class선택

    .class{속성:값;}

  • Group Selector(,) : 선택자를 여러개 그룹으로 선택

    E , #id , .class{속성:값;}

  • [attribute] : 어트리뷰트을 가진 HTML 엘리먼트가 있으면 선택

    [attr]{속성:값;}

  • [attribute=value] : 어트리뷰트의 값이 완전히 같으면 선택

    [attr="value"]{속성:값;}

  • [attribute^=value] : 어트리뷰트의 값이 시작하는 것과 같으면 선택

    [attr^="value"]{속성:값;}

  • inside Selector ( ) : 하위요소 검색

    E E{속성:값;}

  • Child Selector(>) : 자식요소 검색

    E>E{속성:값;}

  • After Selector(+) : 다음요소 검색

    E+E{속성:값;}

가상 셀렉터(Selector)

  • Selector:checked : 체크되면 선택

    E:checked{속성:값;}

  • Selector:disabled : 비활성화되면 선택

    E:disabled{속성:값;}

  • Selector:focus : 포커스되면 선택

    E:focus{속성:값;}

  • Selector:hover : 마우스가 올라가면 선택

    E:hover{속성:값;}

  • Selector:first-child : 첫번째 자식요소 선택

    E:first-child{속성:값;}

  • Selector:nth-child(number | odd | even | n) : 자식들의 순서로 선택

    E:nth-child(순서식){속성:값;}

  • Selector:read-only : 읽기전용일때 선택

    E:read-only{속성:값;}

  • Selector::before or Selector:before : 가상요소를 선택된 엘리먼트 컨덴츠의 앞에 생성하여 선택

    E:before{속성:값;}

  • Selector::after or Selector:after : 가상요소를 선택된 엘리먼트 컨덴츠의 뒤에 생성하여 선택

    E:after{속성:값;}

박스모델 , 인라인박스모델

  • Box-Model
  • Inline-Model

박스사이징(Box Sizing)

  • 디스플레이(Display)

    디스플레이는 요소의 특성을 바꾼다.

    주속성 : none , block , inline , inline-block , flex

  • 비지빌리티(visibility)

    비지빌리티는 요소가 보이는가 안보이는가만 바꾼다.

    주속성 : hidden , visible

가로값(width) , 세로값(height)

  • 대부분은 "px","%","em","rem"을 쓴다.
  • 앞에 "min-" 이나 "max-"을 붙일수 있다.
  • "box-sizing" 여부에 따라 범위가 틀려진다.

  • 패딩(padding)

    배경을 영향을 받는 border-edge 내부 영역.

    요소들 끼리 겹치는 경우는 없다.

  • 마진(margin)

    배경의 영향을 안받는 border-edge 외부 영역

    요소들 끼리 겹치는 경우가 있다.

  • 보더(border)

    컨덴츠와, 패딩 이후의 내곽선

    border-radius라는 속성으로 둥글게 만들수도 있다.

    width값에 영향을 준다.

    요소들 끼리 겹치는 경우는 없다.

  • 아웃라인(outline)

    보더 밖으로 선언되는 외곽선

    outline-offset 이라는 속성으로 더 멀리 보낼수 있다.

    width값에 영향을 안준다.

    요소들 끼리 겹치는 경우가 있다.

오버플로우(Overflow)

컨덴츠가 보이는 영역에서 벗어날때 처리

scoll-x , scroll-y 같은 식으로 가로,세로도 별도 가능.

폰트명(font-family)

  • 기본값 : 브라우저의 기본 폰트
  • 상속성 : 있음
  • 작성방법 : font-family: (generic-family | family-name)+ | initial | inherit;

참고사항

  • "한글"로 된 폰트명이거나, "띄어쓰기"가 있는 폰트명은 ""(따옴표)로 감싸서 선언합니다.

제너릭명?

serif 곡선을 주로 사용하여 작성된 폰트, 삐침이 있다라는것이 특징. (예 : Times New Roman , Georgia)
sans-serif 직선을 주로 사용하여 작성된 폰트, 삐침이 없다라는것이 특징(예 : Arial , Verdana)
monospace 곡선과 직선을 혼합하여 작성된 폰트, 쉐리프와 산쉐리프의 중간정도. (예 : Courier New , Lucida Console)
cursive 필기체(Handwriting)를 기반으로 작성된 폰트
fantasy 장식체(Display) 글꼴을 기반으로 작성된 폰트
예제 : https://fonts.google.com/

색상(Color)

  • 상속성 : 있음
  • 작성방법 : color: HEX(#RRGGBB) | String | RGBA | initial | inherit;

크기(font-size)

  • 기본값 : medium
  • 상속성 : 있음
  • 작성방법 : font-size:medium|xx-small|x-small|small|large|x-large|
    xx-large|smaller|larger| length |initial|inherit;

참고사항

  • smaller , larger는 부모의 사이즈에 따라 작아지거나, 커지는 상대적인 사이즈입니다.
예제 : [CSS2주차] font-size 예제보기

굵기(font-weight)

  • 기본값 : 부모상속됨
  • 상속성 : 있음
  • 작성방법 : font-weight: normal|bold|bolder|lighter|number;

참고사항

  • bolder , lighter는 부모의 굵기에 따라 굵어지거나, 얇아지는 상대적인 사이즈입니다.
  • number는 100단위로 입력하고, 폰트,브라우저 에 따라 굵어지는 값이 다릅니다.
  • number의 "400"은 "normal" 선언과 같고, "700"은 "bold" 와 같습니다.
예제 : [CSS2주차] font-weight 예제보기

참고사항

  • baseline을 기준으로 정렬되는것이 중요합니다.
  • baseline이 없는 한글은 폰트의 영문자의 baseline을 따라갑니다.

데코레이션(Decoration)

  • 기본값 : none (단 HTML의 <a>는 underline)
  • 상속성 : 없음
  • 작성방법 : text-decoration: none|underline|overline|line-through;
none 선 없음
underline 텍스트의 아래에 선을 긋습니다.(링크선)
overline 텍스트의 상단에 선을 긋습니다.
line-through 텍스트의 중단에 선을 긋습니다.(삭제선)

참고사항

  • <a> 링크에 있는 밑줄은 text-decoration:underline 입니다.
  • ":hover" 가상선택자와 함께 사용하여 표현을 많이 합니다.
예제 : [CSS2주차] text-decoration 예제보기

텍스트 변형(text-transform)

  • 기본값 : none
  • 상속성 : 있음
  • 작성방법 : text-transform: none|capitalize|uppercase|lowercase;
capitalize 첫문자는 대문자로, 나머지는 소문자로 표기합니다.(camel 표기법)
uppercase 영문자를 대문자로 표기합니다.
lowercase 영문자를 소문자로 표기합니다.

참고사항

  • 영단어의 띄어쓰기를 기준으로 변화되는걸 주의하세요.
예제 : [CSS2주차] text-transform 예제보기

글자 줄바꿈 (white-space)

  • 기본값 : normal
  • 상속성 : 있음
  • 작성방법 : white-space: normal|nowrap|pre|pre-line|pre-wrap;

참고사항

  • "normal" 과 "nowrap"을 제외한 값은 "IE"에서 지원을 하지 않아서 거의 사용하지 않습니다.[참고]
예제 : [CSS2주차] white-space 예제보기

단어 줄바꿈 (word-break)

  • 기본값 : normal
  • 상속성 : 있음
  • 작성방법 : word-break: normal|break-all|keep-all
break-all CJK(중국어,일본어,한국어)을 "제외한" 언어에서 영역을 넘어갈때, "강제로" 줄바꿈 합니다.
keep-all CJK(중국어,일본어,한국어)에서 영역을 넘어갈때, "단어기준으로" 줄바꿈 합니다.

참고사항

  • word-break 속성은 CJK(중국어,일본어,한국어)가 "아닌" 언어을 기준하여 줄바꿈 기준을 적용합니다.
  • 대부분, 영문자를 계속 연결해서 쓰면, 하나의 단어로 인식해서 줄바꿈이 되지 않고 표현될때 사용합니다.
예제 : [CSS2주차] word-break 예제보기

레이아웃(Layout) 이란?

CSS의 레이아웃은 총 3가지+ 숨겨진 1가지 방법이 있지만, 대부분은 3가지로 합니다.

  • 플로트(Float)

    요소의 플로우(flow)을 조정하여 레이아웃.

  • 포지션(Position)

    요소의 좌표(direction)을 조정하여 레이아웃.

  • 플랙스(Flex)

    요소의 방향축(axis)을 조정하여 레이아웃.

  • 숨겨진 1가지

플로트(Float) 레이아웃

요소의 플로우(flow)을 조정하여 레이아웃하는 기법

플로우(Flow) 란?

흐름(flow)라는 말대로, 좌->우 , 상->하 같은 기본 흐름을 조정하여 적용하는것

플로우 흐름 이미지

float

  • 기본값 : none
  • 상속성 : 없음
  • 작성방법 : 'float: none|left|right|initial|inherit;'
값 설명
none 플로트를 해제합니다.
left 요소를 좌측에 배치합니다.
right 요소를 우측에 배치합니다.

clear

  • 기본값 : none
  • 상속성 : 없음
  • 작성방법 : 'clear: none|left|right|both|initial|inherit;'
값 설명
both left 및 right 을 동시에 해제합니다.

haslayout 이란?

has layout 이미지.

  • float:left|right
  • display:inline-block
  • width:'auto'외의 값 | min-width , max-width 동일
  • height:'auto'외의 값 | min-height , max-height 동일

  • overflow-x,overflow-y:hidden | scroll | auto(ie7만 적용)

포지션(Position) 레이아웃

요소의 좌표(direction)을 조정하여 레이아웃하는 기법

  • 절대적(absolute)

    요소의 공간을 공중에 띄웁니다.

    가로값(width)가 auto 이지만 inline-block의 성격으로 변화합니다.

  • 상대적(relative)

    요소의 공간은 유지한 채 좌표에 따라 움직이게 합니다.

    가로값(width)가 auto 이지만 block의 성격으로 유지됩니다.

좌표(direction)란?

총 4개의 방향을 가지고 있고, 4개의 방향의 선언에 따라 요소를 배치합니다.

position

  • 기본값 : static
  • 상속성 : 없음
  • 작성방법 : 'position: static|absolute|fixed|relative|initial|inherit;'
