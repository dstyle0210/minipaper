흐름이란? 단위 : rem 소수점 한계점 native style sheet (네이티브 스타일 시트)

셀렉터(Selector) - Attribute

  • [attribute]

    [class]{color:red;}

  • [attribute=value]

    [class="test"]{color:red;}

  • [attribute|=value]

    [class|="test"]{color:red;}

  • [attribute^=value]

    [class^="test"]{color:red;}

  • [attribute~=value]

    [class~="test"]{color:red;}

  • [attribute*=value]

    [class*="test"]{color:red;}

  • [attribute$=value]

    [class$="test"]{color:red;}

디스플레이(Display) : Edge.

부모상속값(inherit)

어따써? div{color:red;} h1{color:#000;} div h1{color:inherit} 자신에게 선언된 검은색이 아니라 부모의 색상을 따라가고 싶다면, 선언하게 되면, 우선순위가 변경되면서 적용이 되죠. .test{color:orange} 로 부모를 오렌지로 바꾸면, 이젠 오렌지로 적용되겠죠.

기본값(initial)

단위(Unit) : 절대적인 단위

  • px : 픽셀(가장 기본적인 사용)
  • pc : 파이카(1pc == 12pt == 1/6in)
  • pt : 포인트(1/72in)
  • cm : 센티미터
  • mm : 밀리미터
  • in : 인치

단위(Unit) : 상대적인 단위

  • em : 부모의 font-size을 1em 으로 적용하며, 부모의 폰트사이즈가 늘어나면 함께 들어난다.
  • % : 부모의 width을 100%로 적용하며, 부모의 width가 늘어나면 함께 늘어난다.
  • ch : "0" (숫자 0) 의 가로값을 기반으로 적용되는 사이즈.
  • ex : "x" (영문 소문자 x) 의 높이값을 기반으로 적용되는 사이즈.
  • rem : root 즉 html의 font-size을 1rem으로 기반하여, 절대사이즈로 적용.
  • vw : window의 viewport width을 기준하여 절대 적용됩니다.(그냥 window width의 1/100)
  • vh : window의 viewport height 기준하여 절대 적용됩니다.(그냥 window height의 1/100)
  • vmin : window의 width 또는 height 중 작은 사이즈의 1/100
  • vmax : window의 width 또는 height 중 큰 사이즈의 1/100
