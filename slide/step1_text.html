텍스트 란?

쉽게 생각하면 HTML에 있는 태그가 아닌 모든 텍스트(글자)을 지칭합니다.

예제보기

폰트명(font-family)

어떤 글자체(폰트)을 사용할것인지를 결정합니다.

  • 기본값 : 브라우저의 기본폰트
  • 상속성 : 있음
  • 작성방법 : font-family: (generic-family | family-name)+ ;

제너릭명?(generic-family) [참고]

serif 곡선을 주로 사용하여 작성된 폰트, 삐침이 있다라는것이 특징. (예 : Times New Roman , Georgia)
sans-serif 직선을 주로 사용하여 작성된 폰트, 삐침이 없다라는것이 특징(예 : Arial , Verdana)
monospace 곡선과 직선을 혼합하여 작성된 폰트, 쉐리프와 산쉐리프의 중간정도. (예 : Courier New , Lucida Console)
cursive 필기체(Handwriting)를 기반으로 작성된 폰트
fantasy 장식체(Display) 글꼴을 기반으로 작성된 폰트

참고사항

  • "한글"로 된 폰트명이거나, "띄어쓰기"가 있는 폰트명은 ""(따옴표)로 감싸서 선언합니다.

색상(color)

텍스트에 적용될 색상을 선택합니다.

  • 기본값 : 브라우저 기본컬러(검은색)
  • 상속성 : 있음
  • 작성방법 : color: HEX(#RRGGBB) | String | RGBA;

참고사항

  • 대부분 색상은 HEX(#RRGGBB)방식을 많이 사용합니다.
  • RGBA방식( rgb(R,G,B,A) )을 이용하여, 투명도를 함께 적용가능합니다. / IE8 이하는 적용되지 않습니다.

굵기(font-weight)

텍스트에 적용될 굵기을 선택합니다.

  • 기본값 : 부모상속됨
  • 상속성 : 있음
  • 작성방법 : font-weight: normal|bold|bolder|lighter|number;

참고사항

  • bolder , lighter는 부모의 굵기에 따라 굵어지거나, 얇아지는 상대적인 사이즈입니다.
  • number는 100단위로 입력하고, 폰트,브라우저 에 따라 굵어지는 값이 다릅니다.
  • number의 "400"은 "normal" 과 같고, "700"은 "bold" 와 같습니다.

좌우정렬(text-align)

텍스트의 좌우정렬을 지정합니다.

  • 상속성 : 있음

참고사항

  • 텍스트를 정렬하는 요소로, 블럭요소를 정렬하지는 못합니다.

상하정렬(vertical-align)

텍스트의 위아래정렬을 지정합니다.

  • 기본값 : "baseline"
  • 상속성 : 없음
  • 작성방법 : vertical-align: baseline | length | sub | super | top | text-top | middle | bottom | text-bottom;

참고사항

  • baseline을 기준으로 정렬되는것이 중요합니다.
  • baseline이 없는 한글은 폰트의 영문자의 baseline을 따라갑니다.
  • baseline을 간단하게 확인하는 방법은 영문 소문자 "x"의 하단라인 입니다.
  • 명확하게 구현된 속성이 아니기 때문에, vertical-align:middle; 외에는 집중하지 마세요.

글자 간격(자간)
(letter-spacing)

글자와 글자사이 간격을 지정합니다.

  • 기본값 : 0
  • 상속성 : 있음
  • 작성방법 : letter-spacing: normal|length;

단어 간격(어간)
(word-spacing)

단어와 단어사이 간격을 지정합니다.

  • 기본값 : 0
  • 상속성 : 있음
  • 작성방법 : word-spacing: normal|length;

참고사항

  • 주로 양수 보다는 음수로의 사용이 많은 속성들 입니다.

자평(line-height)

여러줄일때의 줄간격을 지정합니다.

  • 기본값 : normal
  • 상속성 : 있음
  • 작성방법 : line-height: normal|number|length;

참고사항

  • number로 선언할때, 요소의 font-size을 기준으로 1로 표현됩니다.

  • 기본값 : 0
  • 상속성 : 있음

참고사항

데코레이션(text-decoration)

텍스트의 꾸밈요소를 적용합니다.

  • 기본값 : none (단 HTML의 <a>는 underline)
  • 상속성 : 없음
  • 작성방법 : text-decoration: none|underline|overline|line-through;
none 선 없음
underline 텍스트의 아래에 선을 긋습니다.(링크선)
overline 텍스트의 상단에 선을 긋습니다.
line-through 텍스트의 중단에 선을 긋습니다.(삭제선)

참고사항

  • <a> 링크에 있는 밑줄은 text-decoration:underline 입니다.
  • ":hover" 가상선택자와 함께 사용하여 표현을 많이 합니다.
예제보기

텍스트 변형(text-transform)

영문 텍스트의 대,소문자 변형을 적용합니다.

  • 기본값 : none
  • 상속성 : 있음
  • 작성방법 : text-transform: none|capitalize|uppercase|lowercase;
capitalize 첫문자는 대문자로, 나머지는 소문자로 표기합니다.(camel 표기법)
uppercase 영문자를 대문자로 표기합니다.
lowercase 영문자를 소문자로 표기합니다.

참고사항

  • 영단어의 띄어쓰기를 기준으로 변화되는걸 주의하세요.

글자 줄바꿈 (white-space)

텍스트의 글자가 요소밖으로 나갈때의 줄바꿈처리를 적용합니다.

  • 기본값 : normal
  • 상속성 : 있음
  • 작성방법 : white-space: normal|nowrap|pre|pre-line|pre-wrap;
normal 엔터(line break) 및 공백(tab , space)을 공백 1칸으로 표시하며, 글자를 기준하여 줄바꿈 됩니다.
nowrap 줄바꿈이 되지 않습니다.
pre 엔터(line break) 및 공백(tab , space)을 그대로 표현합니다. 영역에 따른 줄바꿈이 되지 않습니다.
pre-wrap 엔터(line break) 및 공백(tab , space)을 그대로 표현합니다. 영역에 따른 줄바꿈이 됩니다.
pre-line 엔터(line break)만 줄바꿈 되며, 공백은 여러개여도 1개만 표시, 영역이 넘어가면 줄바꿈이 됩니다.
예제보기

단어 줄바꿈 (word-break)

텍스트의 단어가 요소밖으로 나갈때의 줄바꿈처리를 적용합니다.

  • 기본값 : normal
  • 상속성 : 있음
  • 작성방법 : word-break: normal|break-all|keep-all
break-all 비아시아언어 에서 영역을 넘어갈때 줄바꿈 합니다.
keep-all 아시아언어 에서 영역을 넘어갈때, "단어기준으로" 줄바꿈 합니다.

참고사항

  • 아시아언어 : CJK(중국어,일본어,한국어)을 의미합니다.
  • 비아시아언어를 계속 연결해서 쓰면, 하나의 단어로 인식해서 줄바꿈이 되지 않고 표현될때 사용합니다.
