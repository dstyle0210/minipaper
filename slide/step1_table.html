표의 특징
HTML태그 중 표(<table>)에 대해 간단하게 알아봅니다.
표,셀 외곽선(border)
표 및 셀의 외곽선에 대해 알아봅니다.
외곽선 간격(border-spacing)
셀(cell)과 셀 사이간격에 대해 알아봅니다.
외곽선 합치기(border-collapse)
셀과 셀사이의 선(border)에 대해 알아봅니다.
표,셀 사이즈(width,height)
표 및 셀에 대한 사이즈를 알아봅니다.
표 레이아웃(table-layout)
표의 레이아웃 기준을 설정합니다.

표(<table>)의 기본형식

HTML의 표에 대해 알아봅시다.

표의 기본 CSS (Native Style Sheet)

table table{display:table; border-collapse:separate; border-spacing:2px; border-color:grey;}
caption caption{display:table-caption; text-align:center;}
tr tr{display:table-row; vertical-align:inherit; border-color:inherit;}
th th{display:table-cell; vertical-align:inherit; font-weight:bold; text-align:center;}
td td{display:table-cell; vertical-align:inherit;}

표 외곽선(border)

표 및 셀에 대한 외곽선(border)에 대해 설정합니다.

  • 기본값 : none
  • 상속성 : 없음
  • 작성방법 : border: border-width border-style border-color;
  • "border-collapse"가 "collapse" 일 경우, "border-radius"속성은 무시됩니다.
  • "border-collapse"가 "collapse" 일 경우, 흐름(Flow)대로 border을 덮어 쓰게 됩니다.
예제보기

외곽선 간격(border-spacing)

외곽선과 외곽선 사이 간격에 대해 알아봅니다.

  • 기본값 : 0
  • 상속성 : 있음
  • 작성방법 : border-spacing: length|initial|inherit;
  • "border-collapse"가 "collapse" 일 경우, 본 속성("border-spacing")은 무시됩니다.
  • length는 "border-spacing:좌/우 위/아래"형식으로 분리하여 선언이 가능합니다.
예제보기

외곽선 합치기(border-collapse)

표와 셀, 셀과 셀사이의 외곽선을 합치도록 합니다.

  • 기본값 : separate
  • 상속성 : 있음
  • 작성방법 : border-collapse: separate|collapse|initial|inherit;
  • "border-collapse"가 "collapse" 일 경우, "border-spacing" , "border-radius" 속성이 무시됩니다.
예제보기

가로(width)

표와 셀의 가로값을 지정합니다.

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : width: length|auto|initial|inherit;

세로(height)

표와 셀의 세로값을 지정합니다.

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : height: length|auto|initial|inherit;
  • "<table>"에 가로값을 선언할경우, 모든 셀의 총합이 됩니다.
  • "<table>"에 가로값이 없는경우, 모든 셀은 "%"가 정상적으로 계산되지 않습니다.
  • "<table>"에 세로값이 있는경우, thead 및 tfoot을 제외한 높이에서 tbody을 비율대로 조정합니다.

표 레이아웃(table-layout)

표의 사이즈의 기준을 설정합니다.

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : table-layout: auto|fixed|initial|inherit;
auto 셀(cell)내의 컨덴츠에 따라 자동으로 가로 및 세로를 지정합니다.
fixed 셀(cell)내의 최초에 지정된 사이즈 값을 기준으로 가로 및 세로를 지정합니다.

표 레이아웃 : 속성(Property) 정리

표,셀 외곽선(border)
border: border-width border-style border-color;
외곽선 간격(border-spacing)
border-spacing: length|initial|inherit;
외곽선 합치기(border-collapse)
border-collapse: separate|collapse|initial|inherit;
표,셀 가로값(width)
width: length|auto|initial|inherit;
표,셀 세로값(height)
height: length|auto|initial|inherit;
표 레이아웃(table-layout)
table-layout: auto|fixed|initial|inherit;
