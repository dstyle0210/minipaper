CSS 초급 : 미디어 : 커리큘럼

미디어의 종류
CSS가 호환하는 미디어에 대해 설명합니다.
미디어의 선언
CSS을 이용하여 미디어를 선언하는 방버에 대해 설명합니다.

미디어의 종류(mediatype)

CSS에서 지원하는 미디어의 종류입니다.

  • all

    전체에 해당하며, 미디어에 상관없이 선언된 CSS 대로 표시합니다.

  • screen

    데이터를 시각적으로 표현하는 장치(display)에 한정적으로 CSS을 적용합니다.

  • print

    데이터를 종이에 인쇄하여 보여지도록 하는 장치(printer)에 한정적으로 CSS을 적용합니다.

  • speech

    데이터를 음성적으로 변환하는 장치(Screen Reader)에 한정적으로 CSS을 적용합니다.

미디어의 선언(@media Rule)

CSS로 미디어에 한정하여 선언하는 방법을 설명합니다.

  • 기본값 : all
  • 상속성 : 없음
  • 작성방법 : @media mediatype { CSS-Code; }
    <link rel="stylesheets" href="CSS경로" media="mediatype" />
