미디어의 선언(@media Rule)

CSS로 미디어에 한정하여 선언하는 방법을 설명합니다.

  • 기본값 : all
  • 작성방법 : <style> @media mediatype and (media feature) { CSS-Code; }</style>
    <style> @import url(CSS경로) mediatype and (media feature) { CSS-Code; }</style>
    <link rel="stylesheets" href="CSS경로" media="mediatype and (media feature)" />

미디어의 종류(Media Types)

CSS에서 지원하는 미디어의 종류입니다.

  • all

    미디어에 상관없이 선언된 CSS 대로 표현합니다.

  • screen

    데이터를 시각적으로 표현하는 장치(display)에 CSS을 적용합니다.

  • print

    데이터를 종이에 인쇄하여 보여지도록 하는 장치(printer)에 CSS을 적용합니다.

  • speech

    데이터를 음성적으로 변환하는 장치(Screen Reader)에 CSS을 적용합니다.

  • 이 외에도 미디어 종류가 있지만, 현재 표준규약상 사라졌습니다. [미디어종류 전체보기]

미디어의 선언(@media Rule)

CSS로 미디어에 한정하여 선언하는 방법을 설명합니다.

  • 작성방법(심플) : mediatype and (media feature)
  • 작성방법(규칙) : ([,|not|only])? (mediatype|(media feature))+ (,|and)? (mediatype|(media feature))?
