레이아웃을 만드는 3가지 방법을 소개합니다.
플로트(float)
요소를 배치하는 방법인 플로트에 대해 설명합니다.
클리어(clear)
플로트을 해제하는 방법에 대해 설명합니다.

포지션(position)

절대적 좌표로 적용하는 방법에 대해 설명합니다.
포지션(position):relative
상대적 좌표로 적용하는 방법에 대해 설명합니다.

새로운 레이아웃 방법인 플랙스에 대해 소개합니다.

레이아웃(Layout) 방식

CSS의 레이아웃방법에 대해 설명합니다.

  • 플로트(Float)

    요소의 플로우(flow)을
    조정하여 레이아웃.

    • float
    • clear

  • 포지션(Position)

    요소의 좌표(direction)을
    조정하여 레이아웃.

    • position
    • top
    • left
    • right
    • bottom

  • 플랙스(Flex)

    요소의 방향축(axis)을
    조정하여 레이아웃.

    • display:flex;
    • flex-direction
    • flex-wrap
    • flex-flow
    • align-items
    • align-content
    • justify-content
    • flex-grow
    • flex-shrink
    • flex-basis
    • flex
    • align-self
    • order

플로우(float)

직역하면 띄우다(float)라는 의미입니다.

  • 기본값 : none
  • 상속성 : 없음
  • 작성방법 : float: none|left|right |initial|inherit;

클리어(clear)

플로트된 흐름을 해제합니다.

  • 기본값 : none
  • 상속성 : 없음
  • 작성방법 : clear: none|left|right | both|initial|inherit;
  • 띄우다(float)라는 의미처럼 "살짝 뜬 요소이다" 라는 개념이 중요합니다.
  • clear는 흐름을 막는 속성입니다. 요소의 float 상태를 해제하려면, "float:none"을 해야합니다.

어려워요 : 해즈레이아웃(hasLayout)

플로트의 영역을 지정해주는 개념입니다. (개념만 이해하세요.)

  • - position:absolute;
  • - float:left|right
  • - display:inline-block
  • - width:'auto'외의 값
  • - height:'auto'외의 값
  • - zoom:'normal'외의 값
  • - writing-mode:tb-rl(ie전용)
  • - overflow:hidden | scroll | auto(ie7만 적용)
  • - overflow-x,overflow-y:hidden | scroll | auto(ie7만 적용)

쉬워요! : 클리어픽스(.clearfix)

플로트의 흐름을 잡아주는 이미 만들어진 클래스 라이브러리 입니다.

.clearfix{*zoom:1} .clearfix:after,.clearfix:before{display:table;content:"";line-height:0} .clearfix:after{clear:both}

속성(Property) 정리

가로값(width)
width: length | auto;
세로값(height)
height: length | auto;
패딩(padding)
padding: length | top right bottom left;
보더(border)
border: border-width border-style border-color;
박스사이징(box-sizing)
box-sizing: content-box|border-box|initial|inherit;
배경(background)
background: bg-color bg-image position/bg-size bg-repeat bg-origin bg-clip bg-attachment initial|inherit;
마진(margin)
margin: length | top right bottom left;
아웃라인(outline)
outline: outline-width outline-style outline-color;
오버플로우(overflow)
overflow: visible | hidden | scroll | auto;
디스플레이(display)
display: inline | block | inline-block | none | flex | table | table-cell;
비지빌리티(visibility)
visibility: visible|hidden|collapse|initial|inherit;
