CSS

CSS는 HTML문서를 꾸며주는 언어입니다.

초급부터 중급까지 강의하며, 꾸준히 연습해보시면서 실무중 간단한 수정이나 급한 상황에서의 지원이 가능하도록 하는것이 목표입니다.

HTML5

HTML5는 새로운 웹문서의 방향을 제시합니다.

HTML5는 초급반만 진행할 예정으로 기존의 HTML이 아닌 HTML5에서 새롭게 정의된 태그만 기준하여 강의합니다.

jQuery

jQuery을 이용해서, 실무에서 사용되는 UI들을 만들어 봅니다.

jQuery를 이용하여 UI을 만들어 보고, UI가 만들어 지기 위한 알고리즘에 대해 토론하고, 새로운 방법에 대해 접근합니다.

NodeJs

NodeJs을 통해서 간단한 웹서버 및 제네레이터를 생성합니다.

NodeJs로 나만의 제네레이터를 생성하고, 간단하게 웹서버를 설정하는 과정까지 진행합니다.
