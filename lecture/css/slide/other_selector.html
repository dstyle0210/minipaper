셀렉터(Selector) : Attribute

HTML태그의 어트리뷰트(Attribute)을 이용하여 선택하는 방법을 설명합니다.

  • [attribute]

    : Attr을 가지고 있으면 선택

    E[attribute]{property:value;}

  • [attribute=value]

    : Attr의 값이 같으면 선택

    E[attribute=value]{property:value;}

  • [attribute*=value]

    : Attr의 값이 "글자로" 포함되어 있으면 선택

    E[attribute*=value]{property:value;}

  • [attribute^=value]

    : Attr의 값으로 시작하면 선택

    E[attribute^=value]{property:value;}

  • [attribute$=value]

    : Attr의 값으로 끝나면 선택

    E[attribute$=value]{property:value;}

  • [attribute|=value]

    : Attr의 값이 "언어형식" 으로 포함되어있으면 선택

    E[attribute|=value]{property:value;}

  • [attribute~=value]

    : Attr의 값이 "단어로" 포함 되어있으면 선택

    E[attribute~=value]{property:value;}

가상 셀렉터(Pseudo Classes) : nth-child(n)

CSS의 가상클래스 중, :nth-child(n)을 좀더 잘쓰는 방법을 설명합니다.

:nth-child(n)
  • "an+b"를 이해하세요.
  • "2n"은 "even"과 같고, "2n+1" 은 "odd"와 같아요.
  • "번째"를 선택하겠다 라는 개념이 중요합니다. ( 1st, 2nd, 3rd, 4th, ..., nth )
  • IE8에서 호환성이 안되기 때문에 주의해야 합니다.
