레이아웃(Layout) 방식

CSS의 레이아웃방법에 대해 설명합니다.

  • 플로트(Float)

    요소의 흐름(flow)을
    조정하여 레이아웃.

    • float
    • clear

  • 포지션(Position)

    요소의 좌표(direction)을
    조정하여 레이아웃.

    • position
    • top
    • left
    • right
    • bottom
    • z-index

  • 플랙스(Flex)

    요소의 방향축(axis)을
    조정하여 레이아웃.

    • display:flex;
    • flex-direction
    • flex-wrap
    • flex-flow
    • align-items
    • align-content
    • justify-content
    • flex-grow
    • flex-shrink
    • flex-basis
    • flex
    • align-self
    • order
예제보기

플로트(float)

직역하면 띄우다(float)라는 의미입니다.

  • 기본값 : none
  • 상속성 : 없음
  • 작성방법 : float: none|left|right;

클리어(clear)

플로트된 흐름을 해제합니다.

  • 기본값 : none
  • 상속성 : 없음
  • 작성방법 : clear: none|left|right | both;
  • 띄우다(float)라는 의미처럼 "살짝 뜬 요소이다" 라는 개념이 중요합니다.
  • 바다에 배가 떠다닌다고 상상해 보면 좀 더 이해하기 쉽습니다.
  • clear는 흐름을 막는 속성입니다. 요소의 float 상태를 해제하려면, "float:none"을 해야합니다.

어려워요 : 해즈레이아웃(hasLayout)

플로트의 영역을 지정해주는 개념입니다. (개념만 이해하세요.)

  • - position:absolute;
  • - float:left|right
  • - display:inline-block
  • - width:'auto'외의 값
  • - height:'auto'외의 값
  • - zoom:'normal'외의 값
  • - writing-mode:tb-rl(ie전용)
  • - overflow:hidden | scroll | auto(ie7만 적용)
  • - overflow-x,overflow-y:hidden | scroll | auto(ie7만 적용)

쉬워요! : 클리어픽스(.clearfix)

플로트의 흐름을 부모영역으로 한정하는(haslayout) 클래스 라이브러리 입니다.

.clearfix{*zoom:1} .clearfix:after,.clearfix:before{display:table;content:"";line-height:0} .clearfix:after{clear:both}

포지션(position)

요소를 좌표로 배치하는 포지션에 대해 설명합니다.

  • 기본값 : none
  • 상속성 : 없음
  • 작성방법 : position: static|absolute|fixed|relative;

  • 절대적(absolute)

    • 포지션을 가진 상위요소를 기준으로,
    • 요소의 공간을 무시하고,
    • 공중에 띄워서,
    • 좌표위치에 놓습니다.

  • 상대적(relative)

    • 부모요소를 기준으로,
    • 요소의 공간은 유지하고
    • 공중에 띄워서,
    • 좌표위치에 놓습니다.

  • 고정(fixed)

    • 루트요소(<html>)를 기준으로,
    • 요소의 공간을 무시하고,
    • 공중에 띄워서,
    • 좌표위치에 놓습니다.
예제보기

포지션(position):absolute

포지션을 가진 상위요소를 기준으로, 요소의 공간을 무시하고, 공중에 띄워서, 좌표위치에 놓습니다.

  • 상위요소 중에 "포지션을 가진 요소가 없으면" 루트요소가 기준이 됩니다.
  • 브라우저의 스크롤에 따라 움직입니다.
  • 공중에 띄워지기 때문에, 형제요소에 영향을 주지 않습니다.
예제보기

포지션(position):relative

부모요소를 기준으로, 요소의 공간은 유지하고 공중에 띄워서, 좌표위치에 놓습니다.

  • 부모요소의 포지션 여부는 상관 없습니다.
  • 브라우저의 스크롤에 따라 움직입니다.
  • 공간이 유지 되기 때문에, 형제요소에 영향을 주게 됩니다.
  • 공간이 유지 되기 때문에, 좌표가 없다면 차이가 없습니다.
  • absolute가 상위요소의 포지션을 필요로 하기 때문에, relative을 이용해서 기준점을 잡아줍니다.
예제보기

포지션(position):fixed

루트요소(<html>)를 기준으로, 요소의 공간을 무시하고 공중에 띄워서, 좌표위치에 놓습니다.

  • 어디에 선언 되었든, 루트요소가 기준이 됩니다.
  • 브라우저의 스크롤에 따라 움직이지 않습니다.
  • 공중에 띄워지기 때문에, 형제요소에 영향을 주지 않습니다.

좌표(direction)

포지션된 요소의 좌표를 설정합니다.

top

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : top: auto|length;

left

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : left: auto|length;

bottom

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : bottom: auto|length;

right

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : right: auto|length;
  • left와 right 처럼 반대되는 속성을 한번에 선언은 가능하지만, 구현이 확실하지 않으니, 사용하지 마세요.

인덱스 깊이(z-index)

포지션된 요소의 인덱스 깊이를 설정합니다.

  • 기본값 : auto
  • 상속성 : 없음
  • 작성방법 : z-index: auto|number;
  • 음수로 적용하면, 아래로 가라앉아서 사라지게 되면서, 포커스도 안잡히게 됩니다.

좌표:하단(bottom)
bottom: auto|length ;
좌표:우단(right)
right: auto|length ;
