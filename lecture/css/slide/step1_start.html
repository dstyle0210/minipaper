CSS란?

CSS(Cascading Style Sheet)는 HTML문서를 꾸미는 언어입니다.

작성법(Syntax)

셀렉터 부분이 있고, 중괄호({}) 안에 속성(Property)와 값(Value)으로 이루어져 있습니다.

셀렉터(Selector) : 선택

CSS의 셀렉터를 이용해서, HTML태그(엘리먼트)를 선택합니다.

  • Universal Selector (*)

    *{property:value;}

  • Element Selector (E)

    E{property:value;}

  • id Selector(#)

    #id{property:value;}

  • Class Selector(.)

    .className{property:value;}

  • Group Selector(,)

    E , #id , .className{property:value;}
  • HTML태그의 "id" 어트리뷰트 와 대입 되는것이 "#" 이고, "class" 어트리뷰트와 대입되는것이 "." 입니다.
  • HTML태그의 "class" 어트리뷰트는 값으로 여러개의 className(클래스명) 이 들어갈 수 있습니다.

셀렉터(Selector) : 검색

CSS의 셀렉터로 HTML태그를 검색해서 선택합니다.

  • inside Selector ( )

    E E{property:value;}

  • Child Selector(>)

    E>E{property:value;}

  • Next Selector(+)

    E+E{property:value;}

  • After Selector(~)

    E~E{property:value;}

컬러(Color)

CSS에서 적용할 색상 작성법 입니다.

  • HEX(Hexadecimal)

    E{property:#RRGGBB}

  • RGB , RGBA

    E{property:rgb(0,0,0) | rgba(0,0,0,0);}

  • 컬러이름.(red , blue 등)

    E{property:ColorName}
  • 대부분의 퍼블리셔는 색상을 적용할때, "HEX 형식"으로 #영문소문자 6자리 로 표기합니다.

단위(Unit)

CSS에서 사용되는 단위의 종류입니다.

  • Pixel(px)

    E{property:numberpx;}

  • Percentage(%)

    E{property:number%;}

  • EM(em)

    E{property:numberem;}
  • 단위는 앞으로 나올 설명에서 "length"와 같아요.
  • "%"는 부모 HTML태그의 사이즈(가로,세로)를 기준으로 적용됩니다.(100% == 부모의 사이즈)
  • "EM"은 부모 HTML태그의 글자크기("font-size")을 기준으로 적용됩니다.(1em == 부모의 font-size)

우선순위(Priority Rules)

셀렉터에는 우선순위가 있어요.

  • 1. inline 직접선언 ( <E style=""></E> )
  • 2. id (#) 로 선언
  • 3. class (.) 로 선언
  • 4. 엘리먼트로 선언
  • 5. 유니버셜 (*) 선언
  • 우선순위가 같으면, 나중에 선언된 속성이 적용됩니다.
예제보기

강제적용(!important)

우선순위가 제일 높은 "!important"입니다.

E{property:value !important;} , <E style="property:value !important;">
  • 값 뒤에 세미콜론(;)없이 "!important"을 입력해야 합니다.

가상 셀렉터(Pseudo Classes)

특정조건일때 선택하는 방법을 알아봅니다.

  • :checked

    E:checked{property:value;}

  • :disabled

    E:disabled{property:value;}

  • :focus

    E:focus{property:value;}

  • :hover

    E:hover{property:value;}

  • :first-child

    E:first-child{property:value;}

  • :nth-child(number|odd|even|n)

    E:nth-child(n){property:value;}
  • 실무에서 많이 사용하는 가상 셀렉터만 정리 했고, 실제로는 엄청 많이 있어요.[참고]

디스플레이(display)

요소의 박스모델 속성을 변경합니다.

  • 기본값 : 엘리먼트형태 상속
  • 상속성 : 없음
  • 작성방법 : display: inline | block | inline-block | none | flex | table | table-cell;
값 설명
inline 텍스트(인라인)요소로 적용합니다.
block 블럭요소로 적용합니다.
inline-block 인라인-블럭 요소로 적용합니다.
none 화면 상으로 완전히 사라지도록 처리합니다.
flex CSS3의 flex 레이아웃 구조의 컨테이너로 적용합니다. (기본은 block속성을 따라갑니다.)
table 테이블 요소로 적용합니다. (기본은 inline-block을 따라갑니다.)
table-cell 테이블 요소의 "td" 또는 "th" 요소로 적용합니다.
예제보기

비지빌리티(visibility)

요소의 보여짐 여부를 결정합니다.

  • 기본값 : visible
  • 상속성 : 없음
  • 작성방법 : visibility: visible|hidden|collapse;
값 설명
visible 화면상 보여지도록 합니다.
hidden 화면상 보여지지 않도록 합니다.
collapse 테이블요소 한정으로 보이지 않도록 처리합니다. 다른요소에 사용하면 "hidden"과 같습니다.
  • "display"와 의 차이점을 꼭 기억하세요.
  • display: 요소의 속성을 바꾼다. / visibility: 보이고, 안보이고만 처리한다.

투명도(opacity)

요소의 투명도를 조절합니다.

  • 기본값 : 1
  • 상속성 : 없음
  • 작성방법 : opacity: number;
  • 0 ~ 1 까지의 투명도를 조절이 가능하고, 소수점은 두자리까지 적용됩니다.
