트랜지션(Transitions) 이란?

요소의 CSS속성을 특정시간 동안 부드럽게 변경해줍니다.

transition

트랜지션(Transitions) 관련 속성을 선언하는 단축속성입니다.

  • 기본값 : all 0s ease 0s
  • 상속성 : 없음
  • 작성방법 : transition: property duration timing-function delay|initial|inherit;
  • 대부분의 트랜지션 선언은 본 단축속성을 이용하게 됩니다.

transition-duration

트랜지션(Transitions)이 적용되는 시간을 지정합니다.

  • 기본값 : 0s
  • 상속성 : 없음
  • 작성방법 : transition-duration: time|initial|inherit;
  • 시간(time)은 "s"(second)와 "ms"(millisecond)로 선언합니다.(1s == 1000ms)
예제보기

transition-property

트랜지션(Transitions) 될 CSS 속성(property)을 지정합니다.

  • 기본값 : all
  • 상속성 : 없음
  • 작성방법 : transition-property: none|all|property|initial|inherit;
  • 속성을 "none"으로 선언하면, 동작하지 않도록 합니다.

transition-timing-function

트랜지션(Transitions) 될 CSS 속성의 변화하는 과정을 설정합니다.

  • 기본값 : ease
  • 상속성 : 없음
  • 작성방법 : transition-timing-function: linear|ease|ease-in|ease-out|ease-in-out|step-start|step-end|steps(int,start|end)|cubic-bezier(n,n,n,n)|initial|inherit;

transition-delay

트랜지션(Transitions)의 지연시간(딜레이)을 설정합니다.

  • 기본값 : 0s
  • 상속성 : 없음
  • 작성방법 : transition-delay: time|initial|inherit;
  • 시간(time)은 "s"(second)와 "ms"(millisecond)로 선언합니다.(1s == 1000ms)

속성(Property) 정리

컨테이너 : display:flex;
플랙스 레이아웃을 사용하겠다 라는 선언에 대해 설명합니다.
컨테이너 : flex-direction
flex-direction: row|row-reverse|column|column-reverse|initial|inherit;
컨테이너 : flex-wrap
컨테이너의 흐름방향(main axis)에 따른 줄바꿈 여부를 설정합니다.
컨테이너 : flex-flow
flex_direction와 flex_wrap을 단축 선언합니다.
컨테이너 : justify-content
컨테이너의 흐름방향(main axis)에 따른 아이템 배치방법을 설정합니다.
컨테이너 : align-items
컨테이너의 정렬방향(cross axis)에 따른 정렬(align)를 설정합니다.
컨테이너 : align-content
컨테이너의 정렬방향(cross axis)에 따른 줄바꿈시 정렬방식을 설정합니다.
아이템 : flex_basis
흐름방향(main axis)에 대한 아이템의 기본값(basis)을 적용합니다.
아이템 : flex-grow
컨테이너에 아이템이 배치되고 남은 공간을 아이템에 분배하는 비(ratio)을 설정합니다.
아이템 : flex-shrink
컨테이너에 아이템이 배치되고 넘쳐진 공간을 아이템에 분배하는 비(ratio)을 설정합니다.
아이템 : flex
flex-grow , flex_shrink , flex_basis을 단축선언합니다.
아이템 : align-self
아이템 개별의 정렬방향(cross axis)을 설정합니다.
아이템 : order
흐름방향(main axis)상 아이템의 우선순위를 지정합니다.
